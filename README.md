# Moving & Learning With Sarah

Welcome to **Moving & Learning With Sarah** — an interactive educational platform designed to help kids learn through movement, games, and fun activities!

## 🎯 About This Project

This website creates engaging educational content that combines:
- **Movement activities** - Get kids moving and active
- **Learning games** - Interactive, fun ways to learn
- **Educational videos** - Age-appropriate learning content
- **Interactive activities** - Hands-on learning experiences

## 📁 Project Structure

```
moving-learning-with-sarah/
├── index.html              # Main homepage
├── css/
│   └── styles.css         # Stylesheet for the site
├── js/
│   └── script.js          # JavaScript for interactivity
├── images/                # Images and graphics
├── videos/                # Video content
├── activities/            # Interactive activities
├── games/                 # Learning games
└── README.md              # This file
```

## 🚀 Getting Started

1. Clone this repository:
   ```
   git clone https://github.com/sidehustles86/moving-learning-with-sarah.git
   ```

2. Open `index.html` in your web browser

3. Start exploring and adding your content!

## 📚 Content Categories

- **Movement Activities** - Physical activities for kids
- **Math Games** - Interactive math learning
- **Reading & Literacy** - Story time and reading games
- **Science Exploration** - Fun science activities
- **Art & Creativity** - Creative projects

## 🛠️ Technologies Used

- HTML5 - Structure
- CSS3 - Styling and design
- JavaScript - Interactivity and games

## 📝 How to Add Content

1. **Add images** → Place in the `images/` folder
2. **Add videos** → Place in the `videos/` folder
3. **Create activities** → Add HTML files in the `activities/` folder
4. **Add games** → Create in the `games/` folder

## 🎨 Customization

Edit `css/styles.css` to change colors, fonts, and layout to match your brand!
